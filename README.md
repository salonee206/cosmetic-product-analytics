# cosmetic-product-analytics
#  Skin Brightness & Hydration Analysis

##  Project Overview
This project analyzes the efficacy of a skincare product in improving:
- Skin Brightness (L*)
- Skin Hydration (Corneometer)

The study was conducted over 14 days with multiple observations.

---

##  Objective
To evaluate whether the product leads to statistically significant improvements in skin parameters.

---

##  Study Design
- Baseline (No product)
- Immediate (Post application - Timm)
- Day 7
- Day 14

---

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- R (Statistical Testing)
- Excel (Data Validation & Analysis)
- Power BI (Dashboard)

## Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Distribution Analysis
- Trend Analysis
- Correlation Analysis
- Outlier Detection

---

##  Key Insights
- Significant improvement in skin brightness (L*)
- Strong increase in hydration levels
- Positive correlation between hydration & brightness
- Majority of subjects showed improvement

---

##  Statistical Analysis
- Paired T-test performed
- p-value < 0.001 → Highly significant results
